# React Gradient View

Inspired from source code of http://www.blackswan.com/dragonfly/

Option:
- **speed**: color change speed in milliseconds - 10 fast, 100 medium or 1000 slow  

```js
import Gradient from 'react-gradient-view';

<Gradient speed={10}>
	You app text / components
</Gradient>
